Yo All!

A heads up to anyone that may be interested in gpsd git head and chrony
or ntpd.  The rest of you can stop reading now.

The ntpd changes are internal, more math is now done to the nSec
rather than to the uSec and proper rounding is being used.

For chrony the socket file has moved and improved.

If gpsd is running as root then gpsd will try to open this socket:
	/var/run/chrony.ttyXX.sock

If gpsd is not running as root:
	/tmp/chrony.ttyXX.sock

Where 'ttyXX' is the GPS device name.  That should satisfy the security
conscious of different flavors and those running multiple GPS.

There will be a few more minor tweaks, but I think this is close to
a releaseable version, once documented.
