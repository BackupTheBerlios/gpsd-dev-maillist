  Hmm, yes, gpsd insists on libusb 1.0 or later.  1.0 is two years old, any
  idea why your OS does not support it?

As far as I can tell 1.0 works only on linux and mac, and maybe on
windows.  It seems no one has been motivated to write the BSD-specific
support because very few things use the 1.0 fork (libusb is very messy
when you dig into the status, it seems - there's another fork openusb
which works on opensolaris).  There's a freebsd version of libusb-1.0 in
freebsd, which seems very odd to me, instead of the main code having
freebsd support (which would then be easy to make work on the other
BSDs).

The list of "projects using libusb 1.0" is pretty short, but maybe it's
incomplete (no gpsd for example):

  http://www.libusb.org/wiki/libusb-1.0
