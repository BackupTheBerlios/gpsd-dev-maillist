points of ngpsd, and we can see if the 2 projects are mergeable

* written using glib using g_io_channels
* it's able to log to a file information (that allows for later
processing)
* will act as a dbus service, sending info every second (or faster,
depending on the gps receiver)
* fully verifies the validity of the nmea sentences before processing
