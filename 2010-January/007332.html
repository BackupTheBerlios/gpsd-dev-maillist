GPSD Folks,

> IPv6 is now fully supported in the base Linux code of GPSD

I updated the Perl library Net::GPSD3 package to support IPv6.  

perl -MNet::GPSD3 -e 'Net::GPSD3->new(host=>q[::1])->watch'

You will need to update Net::GPSD3 to version 0.11 from http://search.cpan.org/perldoc?Net::GPSD3

Thanks,
Mike
