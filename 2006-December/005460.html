GPSD folks,
 I'm interested in integrating a gpsd feed with a wifi broadcast capability for my carputer.  What I'm thinking about is setting up a broadcast with a binary position message and broadcast it every second or so over the wifi.  I can then set up a listener to receive any others broadcasting their position information.  Does anyone on the list have any ideas for standards to start with? Any ideas?  I think a client like netstumbler with a link to a map is what is needed.
