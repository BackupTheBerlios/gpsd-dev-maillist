Michael R. Davis <mrdvt92 at yahoo.com>:
> I'm planning my upgrade to FC5.  I was going through my files and
> noticed that my /etc/rc.d/init.d/gpsd script was not part of the
> /etc/rc.d/init.d/gpsd gpsd rpm.  Any chance you want to add this to the
> distribution?

Probably not.  I used to ship something like this, but the FC5 RPM
already does something better, It installs a hotplug script that
launches gpsd whenever a serial device of a type that might be a
GPS goes active.  This is better than having gpsd always start at boot
time -- if no USB ports light up, not a single cycle is ever spent
on gpsd.
