I found a workaround.  The basic problem is that there are two certs
with the same serial, but it only causes problems if your browser knows
about both of them.  I had the old certificate stored (since it was
signed by an unknown issuer; I don't have the Berlios CA as a trust
anchor).  In firefox's preferences, I just deleted the old certificate.
Then I could click to add the new one as an exception.
