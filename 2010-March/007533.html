I object to checking in generated files in the place where they should
be generated.  If a build platform (as opposed to a run platform!) can't
run tools that are part of our build system, then we should have a
directory with pre-built versions that *only that platform* copies into
place instead of building via some Makefile.am rules that substitute a
copy from running the tool.

Then, we can have 'make check' or 'make dist' also diff the newly
rebuilt file with the cached file.
