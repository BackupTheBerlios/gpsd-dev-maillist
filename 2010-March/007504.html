Hi All,

Just to inform you that I posted a patch on Berlios that builds the
equivalent of libgpsmm & libgps for Qt as libQgpsmm.
The patch adds a subdirectory ("libQgpsmm") with a Qt project and patches a
number of files as to make them conditionally use Qt's QTcpSocket and
compile as C++ code.

The main (and only) purpose of this is to have a client library for gpsd
running under windows/mingw (and OSX, but I guess gpsd runs natively on it?)
The interface is still the same as libgpsmm, so should be source compatible
with existing code, thus normally would allow code compiled under Linux to
use the stock gpsd libraries.

Not being accustomed to gpsd code, it would be great if someone
knowledgeable could review and comment on the patch.

Regards
