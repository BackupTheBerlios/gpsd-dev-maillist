considering that about 5% of computer users might be using a linux desktop,

1) Potlach, using flash
2) JOSM, using java (ok, I think it's open-sourced, now, but I've a personal
dislike of java)
3) Merkaartor (my project), using Qt which is open-source and cross-platform

So, yeah, I guess we have to compromise to make the world better ;-)
