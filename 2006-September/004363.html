> Hi im getting this error message as well.. Im trying to use the PPS feature

Garmin GPS-18/LVC does not support RTCM.  So parity error or not it will do
nothing for you.  Just skip the RTCM.

> Also im getting only a few PPS signals..
> gpsd: carrier-detect on /dev/ttyS0 changed to 1
> In fact i only get 6...sometimes..none..and sometimes..4..near the very
> beginning when i ran gpsd..

Recent versions are hit and miss with Garmin PPS.  Try the newest SVN
and change this line in drivers.c:

#define PUT_MAX_OFFSET       500000          /* microseconds for lost lock */

to this:

+#define PUT_MAX_OFFSET 600000          /* microseconds for lost lock */

Then you should be fine.

We are negotiating a final solution to this, so stay tuned.
