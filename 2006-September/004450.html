Not having troubles here but I am working on it.  Restarting gpsd a
few times gets it going for me.

Can you please run this up to absurd debug level and then email me the
results?  "-D 9" should do it.  No need to annoy the list with that.

You could try this before gpsd:
	stty 9600 raw < /dev/ttyXXXX

As far as I can tell the garmin binary is only at 9600 8N1.  If I set
the Garmin NMEA to 19200, then go to Garmin Serial binary the unit goes
to 9600.  Changing back to Garmin NMEA (only) also sends the speed back
to 19200.  This speed jumping complicates things...

Also be sure you have the latest SVN.

I have a tool to change the GPS from NMEA to Garmin Binary and back, but
it it not quite ready for a release.  Not having to move the unit to a
WinXX box speeds things up.
