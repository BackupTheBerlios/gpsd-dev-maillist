Alternatively, you could have a central place where you 'schedule'  
runs of the regression tests. Bots can connect to the central  
scheduler periodically to determine if it needs to run a regression  
test. Once run each bot can then check in to the central scheduler  
giving its results. All of this could be done via HTTPS.
