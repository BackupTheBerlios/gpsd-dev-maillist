Hi,

I have some problems with svn versions:

xgps does not show a sky-view, or list of satellites.  It shows the 
position etc at the bottom.  xgps from 2.39 does show satellites and 
sky-view, running vs the current gpsd.

cgps takes ages (about 8 s) to start.  This is the same with cgps from 
2.39 running vs the current gpsd.  Running vs an 2.39 gpsd server, the 
2.39 cgps starts instantly, while the new client is slow.

Summary:

cgps:

client \ server:   svn    2.39

svn                slow   slow
2.39               slow   fast

xgps:

client \ server:   svn    2.39

svn                blank  blank
2.39               view   view

This was all with an globalsat EM-406A in NMEA mode.  Does not seem any 
better in sirf mode.
