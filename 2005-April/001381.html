Why not use i18n stuff?
On my machine /usr/share/i18n/locales hold a number of files for each country.
Each file contains info on currency, number and date format and much more 
including measure unit:

LC_MEASUREMENT
measurement    1
END LC_MEASUREMENT

this is for metric, for US measurement is 2
