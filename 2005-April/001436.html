Hi all,
this is my first post on this list.

To introduce myself I bring with me a pair of patches ;-)
but sorry, they are against 2.21... (I don't think it will be much work 
to port them to the 2.22 version)

The first patch adds a real callback function to the library. When 
called, the gps_set_callback() function creates a thread and run 
gps_poll until the thread is canceled with gps_del_callback().
I also fixed return values in gps_poll, but they should be fixed in 2.22 
too.

The second patch deals with C++ language. I wrote down a very simple 
wrapper to the C library, so integration of C++ applications with libgps 
should be quite simple.

Hope you can find this useful.

Cheers,
Alfredo Pironti
