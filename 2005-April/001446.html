> works for me in the case of the CrystalFontz LCD...

Hmm.  Then, here's what I think we need -- a file of udev rules with
the following characteristics:

1) One rule for each product in gpsd.usermap, plus whatever ones you know
   about.

2) Creates each device with owner nobody and group nobody and mode 066.

3) Does *not* create a symlink with a fixed name -- we might have 
   multiple devices.

To go with this, I can change the installation scriptlet to make gpsd
owned by nobody with group nobody, then set its suid and sgid bits.
This should get us to running as non-root.

Does that sound right?  Since you evidently know udev, would you be 
willing to write and test this?
