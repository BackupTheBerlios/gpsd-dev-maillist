> What command string, if any, is needed to enable GPGZDA?
Each NMEA command may be enabled separately by a following string sent to the 
device:
$PMOTG, ccc, yyyy*CC<CR><LF>
where ccc is last 3 chars of command (as GGA), yyyy is the update period in 
seconds (0...9999s, zero to swtch off ), CC is checksum,  asterisk and 
checksum are optional.
$GPGGA comes first in each cycle, GPZDA comes last and it seems that is often 
lost due to lack of time.
In this device, the update rate was set to 1s.
