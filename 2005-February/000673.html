Hello!

Recently I installed gpsd-2.12 from RPMs. It works OK but xgps does not 
recognize --speedunits and --altunits options:

wk at jawor ~]$ xgps --speedunits kph --altunits meters dab.ire.pw.edu.pl
xgps: invalid option -- -
usage:  xgps [-?hv] [--speedunits {mph,kph,knots}] [--altunits {ft,meters}]
[server[:port]]

-speedunits does not work too.

Maybe shortening the options to -s and -a will make them working?

I propose another solution by adding new options (without values):
-m   - setting metric measure system (kph, meters)
-i     - seting imperial/us measure system (mph,ft)
-n    - seting nautical/aero measure system (knots,ft)
