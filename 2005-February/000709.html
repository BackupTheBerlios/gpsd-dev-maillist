The new garmin driver is binary only on USB only.  Unlike other USB
conected GPSes the Garmin really uses the USB protocol and is not just
an RS-232 to USB converter.

Before any data is returned from a Garmin USB device two special binary
enable sequences must be sent to the correct USB device.  One to set the
driver mode and one to enable the GPS.

Not hard, just non-trivial and non-standard.
