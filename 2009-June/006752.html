Make it less complicated. Compile

main()
{
	printf("Hello, world\n");
}

If that works, there is a problem with the gpsd specific build/compile.

If it does not work (as I expect) you have a problem not related to gpsd. The 
likelihood to get an answer here is probably slim.

IIRC, I had that same problem about 5 years ago. My solution was to use a disk 
image from the uclibc site and nfs mount that on my wrt54g. That gave me a 
native development environment. With 200MHz the cpu was plenty fast enough for 
compiling any reasonable app.
