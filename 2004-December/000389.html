Hello!
I had some problems with POSTIN scriptlet of gpsd 2.5
I had to add a semicolon and terminating "fi" to get it running:
....
if [ -f /etc/udev/rules.d/ ]; then echo 'KERNEL="ttyS0", SYMLINK="gps"' 
>/etc/udev/rules.d/80-gpsd.rules
fi

Otherwise I got the error:
line 5: syntax error: unexpected end of file
and the file in udev was not created.
