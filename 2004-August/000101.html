Libtool is disgusting.  I have ripped out the uses of it in the new 
build machinery.  (Now I understand your "not necessarily a feature"
remark, Derrick.)

Why do I say this?  Well, let's start with the fact that it creates an 
invisible subdirectory full of mysteriously named thingoids without
which gpsd won't run.  Then it messes with something else so that the
name of gpsd changes to lt-gpsd in process lists.  

Thank you, I don't care for build tools that pull this much weird, 
random shit behind my back.  There has to be a better way.
