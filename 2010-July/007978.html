  Hello again,

  on second though I decided that it will be more convenient for
me to fork a gpxlogger2. I don't have a park of all operating
systems supported by the GPSD project to test my changes. And my
desired features are possibly not for a command line debugging
tool.

  So I took out gpxlogger.c and hacked on it. I brought the file
to the BSD style(9) indentation, removed unneeded for me dbus
support, and unrolled several functions. I fing it more easier
to read and laconic.

  If someone finds added features (daemonization, syslog, suppressing
logging for a small movements) they can be easily backported to
the core gpxlogger.

  I am going to commit this gpxlogger2 to FreeBSD ports, so if
someone finds it useful, it is ready to use.
