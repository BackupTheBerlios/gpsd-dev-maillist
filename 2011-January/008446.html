Are we not using a package to encode JSON?  The Perl JSON lib is written in C and is licensed under Perl/artistic but now sure how that would goes to BSD.

See: http://search.cpan.org/perldoc?JSON::XS and the code at http://cpansearch.perl.org/src/MLEHMANN/JSON-XS-2.3/XS.xs

Here's an example

perl -MJSON=encode_json -e 'print encode_json({array=>[1.1, 1.2, 1.3]}),"\n";'

{"array":[1.1,1.2,1.3]}
