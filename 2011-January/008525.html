I'm not sure I fully understand this. It looks like your "passthroughs" are
data members in a struct that are mapped directly from JSON elements.

If this is so, you have done exactly what a client binding is not
supposed to do - you have tied the client API to details of the wire
protocol.  Any code relying on "passthroughs" should be considered
vulnerable to breakage without notice.
