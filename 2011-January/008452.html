Gary,
> JSON has no hash structure:

Perl calls it a "hash".  JSON calls it an "object".  It's a key/value pair in any language.

From: http://www.json.org/
JSON is built on two structures:
 - A collection of name/value pairs. In various languages, this is realized as an object, record, struct, dictionary, hash table, keyed list, or associative array. 
 - An ordered list of values. In most languages, this is realized as an array, vector, list, or sequence. 

Example:

perl -MJSON=encode_json -e 'print encode_json({p5=>1.1, p12=>1.2, p16=>undef, p22=>"str", p3=>22/7, p7=>[1,2,undef,4], p9=>{qw{a 1 b 2 c 3}}}),"\n"'

{"p5":1.1,"p3":3.14285714285714,"p7":[1,2,null,4],"p22":"str","p16":null,"p12":1.2,"p9":{"c":"3","a":"1","b":"2"}}

Getting data from here to there is my bread and butter.
