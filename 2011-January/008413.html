There are other chips too, in particular, I use the Venus 634lpx chip
(10hz, SBAS, AGPS) and the docs are available for all the commands as
is a 7-day AGPS binary blob download from an FTP site, however the
internal format is NOT documented.
You can get (and I think set) Ephemeris data too.
http://www.sparkfun.com/products/9758
http://www.sparkfun.com/products/9133
http://www.sparkfun.com/products/9171

And the company now has a GLONASS/GPS chipset for the Venus 7 series:
http://www.skytraq.com.tw/products.htm
I don't know what is required to support GLONASS, but that should be appearing.

And yet another chip can be used for precision GPS since you can pull
the phase information.
http://gpspp.sakura.ne.jp/rtklib/rtklib.htm
Different project, but data comes from a gps chip.

And MTK chipsets have their own AGPS server and commands.

Is all this which you are putting in going to work cleanly with the
other chipssets?
