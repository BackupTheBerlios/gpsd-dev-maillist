Yo All!

I think the subframe JSON may be in its final form.  There are now two
modes of subframe output.  Similar to the AIS output, there is a raw
(scaled=false) and a scaled (scale=true) set out outputs.  The scaled
subframes match the output documente din IS-GPS-200E.  The unscaled are
best for regressions as they avoid floating point comparison issues and
will be usefull for any consumer that prefers the raw data.

You can see the unscaled output this way:

gpspipe -Sw | fgrep SUB

Not sure what the easiest way to see the scaled output is yet.

Anyone with an opinion please speak up before this gets set into the
documentation and regression tests.
