I needed to have formattable timestamp output from gpspipe -t option, so 
I modified gpspipe.c adding the chance to format timestamp just by doing 
gpspipe -t %s (or the standard formatting patterns).
The update is based on this function 
http://www.opengroup.org/onlinepubs/007908799/xsh/strftime.html.
Here is the patch file. I don't know if this is what is necessary to 
include my improvement. Tell me if I made a mistake.
