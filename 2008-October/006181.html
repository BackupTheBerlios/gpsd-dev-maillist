Here is my current autoconf/autoamke cleanup patch; please disregard all
previous ones.

Changelog:

Find athena widgets not only as Xaw but also as Xaw8.

Define HAVE_XAW if found and conditionalize xgpsspeed build on that.
Drop motif libary from link of xgpsspeed.

When checking for ncursse.h, define HAVE_NCURSES_H so that programs that
use that will build on systems without curses.h.  Substitute
NCURSES_LIBS.

When calling mkgmtime_test, don't assume . in PATH.
