I, too, use the init scripts, along with hotplugging USB GPS receivers
automatically, which works quite nice most of the time (except for
one of the devices, see the report on this mailing list). One way
would be to add the name to the supplied device names in some way,
that'd work "most of the time", but would be kind of an ugly solution
IMHO.

Or maybe just stay with sending the serial device names... They're
already there, they're unique (as long as tupled with the hostname).
They, though, are not a meaningful name, possibly reflecting the GPS
brand or stuff like that.
