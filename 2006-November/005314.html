I think that cgps and xgps has to ask for GPS type in some regular interval
otherwise they can report incorrect GPS type. In the case that cgps or xgps is
the first client that connects to gpsd it reports unknown or "?".

Try this:
1) start gpsd -N /dev/gps
2) start cgps or xgps
3) check GPS type
