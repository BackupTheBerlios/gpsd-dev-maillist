> Heads up, I'm working on a big patch. Possibly too big for 2.34.

I'm thinking about one too.

I don't really like the concept of a gps going online and offline and
closing the device and so forth. Or, I don't like it in nowait mode.
To me, nowait is a control for an embedded user or power user to say
"trust me, there is a gps here, there always will be a gps here, even
if it doesn't look like it."

To that end, I'd like to propose that nowait disables the check for a
gps going offline.

Comments?
