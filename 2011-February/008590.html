accumulate $PASHR binary packets - probably ckuethe, I don't
think I did it. 

That means the first thing to do is check to see if the packet sniffer
chunks that file the same way on your machine that it does on mine.

The enclosed file is the output of 

    gpscat -t test/daemon/ac12_binary.log >LOG

Please run that command on your system and look for differences.  That
should bisect the problem neatly; either there will be differences (in which
case the problem is in the sniffer) or there won't be (in which case the
problem is at some later stage).
