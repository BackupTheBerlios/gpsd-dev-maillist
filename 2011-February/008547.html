I wrote a Perl test harness that can simulate a v2 protocol GPSD on whatever port you want to.  You can run as many of then as you want with different data providers.

It's at http://search.cpan.org/dist/Net-GPSD-Server-Fake/ it should not take much effort to update it to protocol v3. 

This one-liner should set up a "fake" GPSD on port 9733

perl -MNet::GPSD::Server::Fake -MNet::GPSD::Server::Fake::Stationary -e 'Net::GPSD::Server::Fake->new(port=>shift)->start(Net::GPSD::Server::Fake::Stationary->new(lat=>38.865826,lon=>-77.108574))' 9733
