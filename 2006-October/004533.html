I am old enough to 'know' that a tab is mod 8.
Remember the 110 baud tty?
Windows 'programmers' tend to set tab to [234]. Reading theire code using a 
real editor pains me.

I have decided for myself to use a convention with the least possible surprise 
for the unsuspecting reader: Indent with spaces only. Indent depth is 4.
No need to save a few bytes by using tabs instead of spaces.

And remember: indent is your friend. Since I am using vim, here my .indentpro 
and .vimrc.

[remco at io ~]$ cat .indent.pro
-kr -cli4 -l96 -nut

[remco at io ~]$ cat .vimrc
set cindent
set shiftwidth=4
set softtabstop=4
set expandtab
set list listchars=tab:»·,trail:·
au FileType make setlocal noexpandtab
