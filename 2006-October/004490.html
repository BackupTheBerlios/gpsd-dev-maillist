Gentlemen,

I found in EverMore specification notice about mapping SBAS sattelites to NMEA
PRN numbers, it is in section describing GPGSA NMEA message. Numbers 32-64 are
in new NMEA specification reserved for SBAS satllites, this is mapping between
SBAS PRN and NMEA PRN:

SBAS ID, SBAS NAME,   SBAS PRN, NMEA PRN
      0  WAAS POR     134       47    
      1  WAAS AOR-W   122       35
      2  EGNOS AOR-E  120       33
      3  EGNOS IOR    131       44
      4  MTSAT-1      129       42
      5  MTSAT-2      137       50

Can gpsd can use it. Chris highligthed several dayes ago, taht gpsd uses 3
SBAS PRN ids for SBAS satellites (xgps for example). I think that this
translation should be used for GPGSA messages generated by gpsd at least...
Mapping is simple, it is:
NMEA PRN = SBAS PRN - 87;
