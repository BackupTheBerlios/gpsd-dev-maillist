Yo All!

I just checked in a new program: garmintool.  It is a standalone tool to
force a Garmin Serial GPS into NMEA or binary mode.  The it autobauds
to the new speed and leaves the serial port properly set.  This allows
the gpsd autobaud to detect the Garmin rapidly.

Changing the operating mode of a Garmin Serial GPS inside of gpsd is
problematic.  Some folks will not want the opertating mode to be change.
Others will be surprised when it is.  When the operating mode is changed
the GPS requires a long reset period and often awakens at a new baud
rate.

By putting the setting of an operating mode into a user tool we give the
user the tool to set the GPS as he wishes.  Whatever mode the tool
sets the GPS to then gpsd acn use it.

Any Garmin Serial users out there please give it a try.

This does not do anything for Garmin USB owners.  Garmin USB devices are
always in Garmin binary mode.

Now that the Garmin Serial binary driver is working there is just a bit
more work to get Garmin USB working in streaming binary mode.  This
is required since the Garmin USB packet mode is broken in the kernel
driver.  The main problem I have now is figuring out how to get the
kernel drive to change modes.
