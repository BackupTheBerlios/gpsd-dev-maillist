I see nothing in that document referring to milliseconds or nanoseconds.

There are a thousand 1023-chip codewords per second. Each code word is
1ms long. Count code words and you have your milliseconds. Each chip
is about 0.9775uS. So if you can count code chips and keep your local
timer properly tuned you get something pretty close to +/- 1uS
