So run sh -x xmlto ..... to see what it does. then write a local
script to do that, or put it in the makefile. If xmlto is just a
(kinda nonstandard) shell script that dispatches other utilities (like
xsltproc) then it should be easy to replace with some makefile targets
invoking those utilities.
