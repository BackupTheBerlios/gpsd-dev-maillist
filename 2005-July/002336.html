Ah, yes. That would be my assumption. For now, SiRF receivers are
seeming like the gps hacker's choice. Though there are a number of
other interesting platforms out there that I haven't got around to
blowing money on...

analog devices -> SST-NAV-2500
motorola -> oncore
navsync -> CW25 (looks a bit like SiRF)
novatel -> superstar2
rf microdevices -> rf8900xemix -> XE1610
rockwell -> jupiter, zodiac
signav -> mg5000
sony -> CXD2951 gps chipset
trimble -> lassen *
ublox/atmel -> antaris
u-nav microelectronics -> uN901
zarlink (mitel) -> GP4020 (orion)

fastrax (www.fastrax.fi) uses the sony chipset in their uPatch
products, and they have the command specs available for download. the
command spec also includes hints on decoding ICD-GPS-200 data
(http://www.fastrax.fi/showfile.cfm?guid=e6a0ea2c-2ff6-4f15-bffa-de0104322c48)
