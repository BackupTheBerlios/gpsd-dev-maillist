Hello,

I was looking for a java application for my mobile phone that will change
mobile to GPS device. I have an external GPS module that talks NMEA and I was
looking for an application to change my mobile to simple GPS. There are
hunderds games for GSM mobiles but I failed to find GPS application... :-(

But I have found something else. Software for PC that will use GSM information
from mobile to track user position. GSM network is based on small cells and
every cell has unique ID. Size of cells differs. Some GSM mobiles could report
visible cell IDs with AT commands. And the whole trick that is used to use GSM
phone as a cheap GPS is to translate cell IDs to "GPS" possition. First, cell
IDs has to be mapped and stored to a DB. Once DBs is created, it is posible to
translate GSM cell ID to position. Sure, accuracy is not so good as with real
GPS but it could be fine, more useful as a game only. One small problem is
here that cells IDs could change in time (BTS replacements, hw upgrades,
etc).

Here is a link to the BTSScan software and pages about it (czech only); There
is a huge DB of BTS in Czech Republic and Slovakia on the web too.

http://www.gsmcables.cz/soft.htm

In theory, new driver for gpsd could be created to use GSM based "GPS". Just a
joke... Or does it make sense?
