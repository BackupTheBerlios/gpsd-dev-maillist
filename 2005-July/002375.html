Eric S. Raymond <esr at thyrsus.com>:
> What's more unfortunate is that the few devices that do report error
> estimates don't document whether they're one- or two-sigma.  I'm computing
> two-sigma error estimates.  Garmin binary devices, Zodiac binary devices,
> and the Garmin PGRME sentence report error estimates, but for *none* of
> these is the confidence interval actually documented.

I've been doing some more digging and it appears that I previously 
misunderstood some of the Web sources on this topic.  Garmins, it
appears, normally report 50% CEP.  Garmin doesn't document this
but sophisticated users have done the experiments.

I can't find any clue what the confidence interval is for the 
error estimates returned by the Zodiac.  Rob Janssen, do you know?

I'm about to commit some changes that will clean everything up to report
uniform 95% confidence levels.  The cost, though, is that we can't
use the Zodiac's reports any more, not till we know that its confidence
interval is.
