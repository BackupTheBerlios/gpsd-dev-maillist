> I recently discovered something strange with the used flag reported

I have a Haicom 204S. I've never seen it do this.  

The Haicoms use SiRF-II chips, so if there is a parsing bug in libgps
it should affect lots of other GPSes as well.

What is xgps? I looked at [2] and couldn't determine whether or not 
it's just a Debian-local name for gps.

If you can capture to a file a sequence of NMEA sentences that exhibits
this bug (that is, one with 4 satellites active but gps only displaying 1), 
the bug could be reproduced using the -l (logfile) driver.  In that case
I have no doubt it would be fixed rather quickly.
