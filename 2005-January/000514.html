I just discovered that the number of satellites used are '[]' before
it is set by anything arriving from gpsd.  I expected a number.  I
believe it is better to initialize this value to zero, instead of an
empty array.  Here is a patch to do this.

Index: gps.py
===================================================================
--- gps.py      (revision 1311)
+++ gps.py      (working copy)
@@ -64,7 +64,7 @@
        self.mode = MODE_NO_FIX
        self.mode_stamp = gps.timestamp(now)

-       self.satellites_used = []               # Satellites used in last fix
+       self.satellites_used = 0                # Satellites used in last fix
        self.pdop = self.hdop = self.vdop = 0.0
        self.fix_quality_stamp = gps.timestamp(now)
