I think you miss what github is really for - easy collaboration. There
is no need to have everything at one place. You have your homepage,
mailing list and whatever else you need at your own site (or on alioth,
savannah, ....) and put your git repository on github to allow easy
collaboration, pulling and merging, together with a network view of all
forks on github. There is no need to migrate *everything* to github.
