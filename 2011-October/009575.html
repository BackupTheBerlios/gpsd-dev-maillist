> I second github, too. Or at least let's have a copy there so using the
> various nice features will be available.

Kind of late - I moved us to Savannah days ago.  I did consider github, 
but rejected the idea for various reasons, including:

1. the github engine is proprietary.

2. IIRC, github uses Google Groups for mailing lists.  Google Groups *sucks*;
having to log in to read mail is bogus.

3. When you're driving as possible to get out of the blast radius
of an explosion, that is no time to buy a fancy new car.

4. Somebody else has 'esr' at github :-).

I'm open to argument that we should migrate to a fundamentally
different kind of forge - possibly github - at our leisure, but we
didn't need the process friction of doing that at the same time as
getting the project out of danger.  The move was hairy enough as
it was - cost me four days of hard work.

Savannah has the virtue of being a lot like Berlios, only with less suck.
That's what we needed, this time - a safe place to land.
