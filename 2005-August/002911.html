I think it's about time to push out another release, if only to get
Chris Kuethe's excellent work on the website into the public view.  I
don't know of any showstopper issues in the way.  If you do, please
tell me.

Top things we're working on:

* The out-of-order GSV problem.  I think I believe Wolfgang's scenario
  that this is a problem with data received while a PPS pulse is
  holding CD high.  Question is, what we're going to do about it...

* Bluetooth devices.  The bluetooth stack closes its serial devices
  when the GPS wanders out of range,  We may need to hack the
  architecture so gpsd tries a device reopen whenever read from GPS
  return EBADF, or something similar.

* 10-second lag from SiRF binary devices.  OK, so this is almost
  certainly a SiRF firmware bug.  Not all firmware versions do it,
  Can we get a list of known-good and known-bad versions together?

I don't consider any of these release blockers.
