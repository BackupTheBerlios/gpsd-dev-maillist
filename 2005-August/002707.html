Yo Eric!

The Garmin GPS seems to be working fine again.  It turned out to be the
new state variable outbuflen that I was not setting.

The double call of garmin_probe() looks to be my fault.  You call
garmin_probe() and if that is successfull you call garmin_init().
garmin_init() calls garmin_probe again.  Seems to have been that way
a long time and no real harm done.   I may have done that to be sure the
garmin wakes up properly after a long sleep.

You can ship it as it currently sits, but if I have more time later
I'll try to clean up the _int()/_probe() thing a bit more.
