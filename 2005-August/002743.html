Yo Eric!

Still a problem with 'unknow sentence's.

When the GPS sends this:

$PGRMO,GSA,1*23\x0d\x0a

Then I set this in the logs:

gpsd: <= GPS: $PGRMO,GSA,1*23
gpsd: unknown sentence: "$PGRMO,GSA,1*23\x0d\x0a"

I think the fix could be small.  Change this:

            retval = 0;         /* unknown sentence */

To this:
	    retval = EMPTY_SET;      /* unknown sentence */

Where EMPTY_SET is a new flag that just means sentence seen.

#define EMPTY_SET  0x40000000u

Uh, oh, hate to use the next to last free flag bit...
