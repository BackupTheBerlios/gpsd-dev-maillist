> You mean, like the N command?

Somewhat, yes. But it should also switch the gps to send all sentences (iirc
there are some vendor specifics that switch off vtg at initialization).

I'd propose the following:
- Enhance 'n'. N=ALL (or something like it) switches to NMEA and turns
everything on.
- Add a command line option to gpsd that connects to the gps (probably with
some retrys), sends some specified commands (eg. B=4800,N=ALL) and exits.
- As switching the baud rate requires user interaction, it may be better to
leave it alone. If the user changed it, he should change it back =)
