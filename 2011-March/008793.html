One thing I realized is that it sounds like you are starting from git on
the embedded/restricted system, rather than using something that's like
a release tarball.

To make a release tarball (with configure and Makefile.in generated,
plus ltmain.sh), which won't need autoconf, automake or libtool to
build, use a computer with all the usual tools installed, and then
./configure gpsd somehow, and then:

  make dist

You'll get

  gpsd-2.96~dev.tar.gz

which can then be unpacked and built.
I think this will make your efforts a lot easier.
