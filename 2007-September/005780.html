"script" is your friend. it is a tool that will capture all
input/output to a tty into a file.

ttyp2# script
Script started, output file is typescript
ttyp4# ./configure <options>
(configure does stuff)
ttyp4# make
(make does stuff, and fails)
ttyp4# exit
exit
Script done, output file is typescript
ttyp2#

then you can mail us the "typescript" file. This should help us figure
out where the problem is.
