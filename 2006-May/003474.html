Hello, I'm the technical lead of an open-source project called gpsd
that provides a monitoring daemon for GPS devices connected to
computers running Linux and other opn-source operating systems.  Among
other places, our software was used as a critical component in a DARPA
Grand Challenge entry.

We've recently become aware of the existence of a master NIST time
offsets file that Network Time Protocol daemons use to get the current
leap-second offset.  We found an old copy at

	    http://www.cis.udel.edu/~mills/leap-seconds.3169152000

but the NTP documentation is silent about where it came from.

Is there a stable public URL to the master version of this file?

We're asking because if we knew where to fetch it reliably, we could
arrange for instances of our daemon to use it rather than a hardwired
constant that has to be tweaked whenever another leap-second is
declared.
