We have had many cases where multiple copies of gpsd and xgps have been
installed at the same time.  For example, if you have gpsd and gpsdrive
BOTH installed then your package manager will report you are running the
newest gpsd, but the really old gpsd from gpsdrive will be first in
you path.

So please humor us and run "xgps -V'.
