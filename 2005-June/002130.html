My intention was to release 2.26 tomorrow, June 15.
The bug list currently has two items:

*** PGRME is not filled in on Garmin USB

*** gpsd with multiple GPSes eats the processor

In addition, there is a discussion going on about data lifetimes that
may lead to some minor changes in behavior on generic NMEA devices.

While the eat-the-processor bug is serious, it only manifests in an
unusual use case.  The data-lifetime discussion will only affect the
20% of non-SiRF devices.  Accordingly, I don't consider these to be
blocking issues.

Gary, I would very much appreciate it if you could find time to chase
and fix the PGRME bug tonight.  This is all that is currently in the 
way of release.
