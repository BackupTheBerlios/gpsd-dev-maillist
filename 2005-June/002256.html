The svn head code no longer relies on the FIONREAD ioctl.  Instead, I
have changed it to use non-blocking file descriptors everywhere.
This change has three effects:

1. It may fix the hang-during-backups bug reported by Rob Janssen.
   Rob, would you check this please?

2. It may break the Garmin driver, which is the only one that doesn't
   use the generic packet getter.  I tried to patch it correctly, but
   can't test the results.

3. It may break communication with a DGPS server.  Again, I tried to
   patch this correctly, but can't test the results.

The new code seems to work fine on my SiRF GPSes.  Please test and
report.
