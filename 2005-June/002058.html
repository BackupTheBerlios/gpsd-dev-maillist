> playback of logs. do i simply `dd` the device for some bytes while

There's an easier way.  The new 'l' command in sirfmon logs packets.

> and what are
> # Pause-noted:
> # Well-behaved:

Pause-Noted should be Y or N as there is or is not a visible pause
between cycles.

Well-behaved should by Y if all sentences in the same cycle have the
same timestamp, N otherwise.

These are described in the Hacking Guide.
