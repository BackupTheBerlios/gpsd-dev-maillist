First benny from having both splint and a regression test -- type
cleanup took about thirty minutes flat.  Mostly introducing speed_t,
making sure all buffer sizes and size arguments are size_t, and using
bools consistently for logical values.

There doesn't really seem to be much more to be done there, shout of
dubious propositions like turning the GPS week and TOW into time_t.
I'm open to suggestions.

Remaining issues:

* Making sure the TSIP driver ain't broke.
* Looking for Zodiac, uBlox, and TSIP packet dumps.
* True North compass support.
* Time zone setting, if possible.

I'd like to shoot for stable release Wednesday, 16 June.  At this
point I consider TSIP support the only blocking issue.  I'll take
True North support if we can get it, but won't cry if it doesn't
come in on time.
