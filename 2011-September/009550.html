> On 09/29/2011 12:49 PM, Jon Schlueter wrote:
> > Attached is a patch which adds comment if you don't have chrpath installed
> > it also allow compilation to complete if you don't have chrpath installed
> 
> Running regressions test from scons should be possible if scons would
> use LD_LIBRARY_PATH or similar quirks to start them, I think. Didn't
> give it a try, though.

>From near the front of SConstruct:

# If chrpath(1) is not available, the RPATH of built binaries won't be
# set so they can see shared libraries in this build directory. Setting
# LD_LIBRARY_PATH like this partially solves this problem, ensuring that
# at least for tests run under scons the local libraries will be found
# anyway.
#
envs = {'LD_LIBRARY_PATH': os.getcwd()}
for var in ('PATH', 'PKG_CONFIG_PATH', 'STAGING_PREFIX'):
    if var in os.environ:
        envs[var] = os.environ[var]
