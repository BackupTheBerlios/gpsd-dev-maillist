On 08/09/2011 17:53, Michael R. Davis wrote:
> Ed,
> It took me a long time to "get" SCons but you don't use the prefix at all.
>  
>> Hi, I was having a bit of a stab at converting the gentoo packaging to
>> use the new SCONS system and gpsd3.x and I'm not sure of the 
>> correct way to install into a prefix directory?
>  
> export DESTDIR=$RPM_BUILD_ROOT
> scons install
>

Aha - perfect, thanks

Note, I examined other build recipes in gentoo and they all use this idiom:

    scons prefix="/" install DESTDIR="${D}"

This means that gpsd is unusual in needing:

    DESTDIR="{D}" scons prefix="/" install

I don't know enough about scons to comment either way, but can someone
consider whether it's common/normal/suggested to support the top idiom
in gpsd?
