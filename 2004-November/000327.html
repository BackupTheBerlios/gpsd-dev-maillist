First of all, I want to say thanks for all the updates done to gpsd over the
last few months.  It really is a great piece of software.

I'm not writing this because I have a problem, but I wanted to throw more data
into the discussion about the GPGSA status vs. the GPGGA mode and the
relationship between them.  I'm using a Garmin 152 GPS (spitting out NMEA
sentences) with gpsd.  By default, this GPS does not send out GPGSA sentences. 
As a result, the "mode" variable would not get set to anything except
MODE_NO_FIX and gpsd would report "GPS is confused about whether it has a fix".
 There is a setting in the main menu where the GSA sentence can be turned on,
but Garmin obviously considers this sentence optional.  I've seen the
discussions in the mailing list archives on this topic, so I know this quite a
confusing issue.  I don't have any answers, but I thought I'd throw my
experience out there for the developers to chew on.  If a discussion erupts
again about this topic, I'll do my best to answer any questions that come my
way.

Thanks again.

Dalen Kruse
