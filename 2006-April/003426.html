   v1.5 APA <- does not work

   v2.1 GSA <- works excellent
 9. A list of NMEA sentences it emits
   Cycle report Fri Apr 21 20:51:30 2006, Generic NMEA, 9600N1, cycle 1s
   The sentence set emitted by this GPS is: GSA GGA GSV GLL RMC
   GSA: is emitted once a second.
   GGA: is emitted once a second.
   GSV: is emitted once a second.
   GLL: is emitted once a second.
   RMC: is emitted once every 1 seconds.
   Send cycle is once per second.
10. Notes on the device
   usb has 3 modes: 
   NMEA data comm (3 submodes): outputs GPS data (creates /dev/ttyACM0),
   USB file transfer: transfer files (creates /dev/sdX and /dev/sdX1),
   Power Only: use usb only for electrical power
