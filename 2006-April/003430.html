this patch fixes several problems with the libgps binary.

* fix warning caused by assignment of long int ti size_t argument

* fix function declaration to match gps_data_t raw_hook pointer type
* define function static to get rid of warning about missing prototype

* fix warning by defining function type as int

* make sure definition of NAN is included from math.h
