Hi,

> +if [[ `glibtool --version > /dev/null` -eq 0 && `uname` = 'Darwin' ]]; 

That's not a correct shell idiom. "glibtool --version > /dev/null"
will print nothing to stdout, thus this test will never fail.

smurf at kiste ~ $ if [[ `glibtool --version > /dev/null` -eq 0 ]]; then echo GO; else echo DIE; fi
-bash: glibtool: command not found
GO
smurf at kiste ~ $ if [[ `libtool --version > /dev/null` -eq 0]]; then echo GO; else echo DIE; fi
GO

Just run

## glibtool --version > /dev/null 2>&1

prior to this line, and check $?.
