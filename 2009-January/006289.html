On Sun, Jan 11, 2009 at 10:21:34PM -0500, Michael Ledford wrote:

> No, I blame the USB specification for  not providing for a serial port 
> (RS-232) like HID descriptor.
It does. It's called CDC-ACM (Communications Device Class - Abstract 
Control Model). See CDC1.2_WMC1.1/usbcdc12/CDC120.pdf and 
CDC1.2_WMC1.1/usbcdc12/PSTN120.pdf in [1].
While it does not support CTS (that part of handshaking probably is 
supposed to be handled by the USB<->serial converter) every other signal 
present on 9-pin RS232 is available.


PS: Getting OT, MFT set accordingly.

[1] http://www.usb.org/developers/devclass_docs/CDC1.2_WMC1.1.zip
