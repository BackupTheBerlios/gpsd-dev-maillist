I'm not sure if this is good XML but, here's my cut at a GML version.  

How do I get gpsd data into a web page?  How can I host my gpsd on the
internet?  The easy way is to use a perl CGI script that leverages
Net::GPSD like this...

[code]

#!/usr/bin/perl -w
use strict;
use Net::GPSD;

my $g=new Net::GPSD;
my $p=$g->get;

print qq{
<?xml version='1.0'?>
<Report xmlns:gml="http://www.opengis.net/gml">
  <items>
    <Item>
      <ID>35643563245</ID>
      <position>
        <gml:Point srsDimension="2"
srsName="urn:ogc:def:crs:EPSG:6.6:4326">
          <gml:pos>}. $p->lat. " ". $p->lon. q{</gml:pos>
        </gml:Point>
      </position>
    </Item>
  </items>
</Report>
};

[/code]

returns...
