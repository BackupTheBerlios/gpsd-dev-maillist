The only official definition for NAN or Infinity I see on BSD systems
is the following and it is a non-starter for array initialization.
(Not to mention having the amusing bug were some devious routine could
write into HUGE_VAL and set it equal to 0.0 for everyone else.)

    extern char __infinity[];
    #define HUGE_VAL	(*(double *)(void *)__infinity)
