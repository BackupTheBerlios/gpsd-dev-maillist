There once was a problem with setting the Reply-To" header to the  
list which involved auto-responders creating mail loops that would  
flood the list. Mailman now detects most forms of auto-responses and  
breaks the loops. However, I did experience one of these loops just  
one year ago on a list I manage for a user group which involved a  
users vacation program that stripped all the headers so mailman  
couldn't detect the loop. Even after that, I still run my lists with  
the Reply-To: header because it keeps the discussions flowing.
