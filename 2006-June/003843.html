Brad,
You can easily use Net::GPSD to script an http based return.

perl -e '
use Net::GPSD;
$g=new Net::GPSD;
$p=$g->get;
print "Context-type: text/xml\n\n<pt>\n".
      "  <lat>". $p->lat. "</lat>\n".
      "  <lon>". $p->lon, "</lon>\n".
      "</pt>\n";'

returns...

Context-type: text/xml

<pt>
  <lat>38.865986</lat>
  <lon>-77.108525</lon>
</pt>
