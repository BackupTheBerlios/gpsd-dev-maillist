I see no error there... that was you sending a SIGINT, while gpsd was
waiting in select. You may wish to build gpsd with "-g", and then run
it from the build directory, but not under gdb. Then when you get the
segfault, you will have a coherent set of {core dump, binary with
debug sysbols, sources} from whence you can generate the backtrace.
