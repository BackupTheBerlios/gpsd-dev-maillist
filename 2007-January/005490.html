Happy new year!

I think it's time to report the current status.

I've started to write a python wrapper for libgps:

* it's possible to connect to a gpsd instance and send queries or poll
  for data
* the current fix is returned as python dictionary
* the Lexer from gpspacket is integrated and a bit improved

I've also experimented a bit to take advantages of the Python language
features.

It's too early to publish a patch. I'll catch up on this later.

Cheers, Andy
