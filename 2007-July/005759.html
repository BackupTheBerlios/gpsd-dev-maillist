I know that I've said this before but I just want to mention it again. 
THE listener address should be a LIST of addresses and ports.  See the
PostgreSQL and Apache configuration files as inspiration.  localhost
::1 and 120.0.0.1 should be the defaults. The outside IP addresses
should not be enabled by default.
