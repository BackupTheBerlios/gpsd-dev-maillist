> If we have a preference how the daemon should be started, we should communicate that. 

I think we have.

RPM
PIDFILE=/var/run/gpsd.pid
CONTROL_SOCKET=/var/run/gpsd.sock
OPTIONS="-n -F $CONTROL_SOCKET -P $PIDFILE"

PKG
DAEMON_OPTS="-F /var/run/gpsd.sock"

HOTPLUG
CONTROL_SOCKET = "/var/run/gpsd.sock"
gpsdcmd = "gpsd -F " + CONTROL_SOCKET


> Fedora has local policy (settings) in /etc/sysconfig

The RPM does not currently support this but it could.
