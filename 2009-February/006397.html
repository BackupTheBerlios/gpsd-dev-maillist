I just pulled the current svn (5150) and the Garmin USB (binary only)
and the Garmin Serial Binary both work for me.  I get nice PVT info.
That is on a GPS18/USB and a GPS18/LVC.

Mode changing is still problematic.

I do see this error in this struct:

struct gps_type_t garmin_usb_binary_old =
[...]
    .configurator   = garmin_usb_configure,     /* eable what we need */


The hardware on Garmin USB does not allow reconfigurations.

And I see the Garmin USB binary changer has been removed which explains why
it does not work for me.

struct gps_type_t garmin_ser_binary =
[...]
    .configurator   = NULL,             /* enable what we need */
