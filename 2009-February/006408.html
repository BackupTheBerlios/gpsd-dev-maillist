Hi all,

here is a patch that adds initial IPv6 support for gpsd (patch against
version 2.38).

It allows the gpsd daemon to listen on a v6 socket, when started with the
"-6" option.
It also updates netlib.c, libgps.c, gpspipe.c, gpsctl.c to allow gpspipe and
gpsctl to connect to the gpsd daemon with IPv6.

It has not been thoroughly tested, but works pretty well for my needs. Some
other changes might be needed for the whole gpsd package to be IPv6 enabled.

I hope it helps... and thanks for developping this great software :)
