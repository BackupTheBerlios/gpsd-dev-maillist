> The creating/deleting of the S44gpsd/K56gpsd files should be handled
> by calling chkconfig. If you run "chkconfig --add gpsd" in the post
> install script, it will add all the start/kill links for you. If you
> run "chkconfig --del gpsd" it the pre-uninstall script, it will
> remove them. You should also run "service gpsd stop" in the
> pre-uninstall script. (If you did not call the file in init.d gpsd,
> change the name in the commands to the name you used.)
