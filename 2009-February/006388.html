Greg Troxel wrote:
> Do we have a script that does search?  I suppose the tough part is to
> decide works/not for each iteration.

git-svn clone --stdlayout svn+ssh://svn.berlios.de/svnroot/repos/gpsd

and use git-bisect in the checked out repository, like

git bisect start
git bisect bad
git bisect good the-last-commit-or-tag-or-branch-you-know-to-be-fine

man git-bisect will tell you the details :)

Deciding if it works or not is still a manual job, though.
