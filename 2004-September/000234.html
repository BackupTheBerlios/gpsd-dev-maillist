Hello!
Today I tried to install gpsd on a machine in my laboratory on Warsaw 
University of Technology. I had some problems with a downloaded RPM, which 
contained no libgps library so I had to rebuild the RPMs with some changes.
The patch file is uploaded onto gpsd page at: 

http://developer.berlios.de/projects/gpsd/

There are two changes:
1) %configure is used as this sets necessary dirs for everthing (no --prefix 
needed)
2) libgps.so.1.0.0, libgps.so.1 and their manpages are moved to main binary 
RPM
