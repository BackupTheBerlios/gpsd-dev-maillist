Russ gave me an account on a machine with a Zodiac Earthmate hanging
on it.  Somewhat to my surprise, gpsd appears to be working nicely on
it; I can open port 2947, start raw mode, and see the NMEA faked up by
the analyze() function.  I didn't see real fix data, but I suspect
that's because the machine is indoors.

That clears the last issue I had before being ready to ship 2.0.  This
is a last call for bug reports and feature requests.
