    Alright, it has been done.  The source can be checked out from 
subversion using instructions provided by berlios:
http://developer.berlios.de/svn/?group_id=2222

    The problem I mentioned earlier is still present.  For usage please 
refer to testread.c in the examples folder and docs below.  The testread 
program demonstrates the bug... be sure to save your work before running as 
it could crash your system.

    `make test' will build the test program within the library.  `make' is 
sufficient to build the test program in examples folder.

Documentation provided courtesy of Doxyfile:
http://svn.berlios.de/viewcvs/*checkout*/ttyproc/html/index.html

    This probably will not be the end all solution, there is still the issue 
of being able to detect when a usb device is (re)plugged in.  Hotplug will 
most likely come into play with that.
