Dirty way:
Add some hardcoded paths where devices are located and try if they exist end
then open them.
In the example on Linux-USB they use /dev/usb/tts/0, my SUSE 9.1 has only
/dev/usb/ttyUSBXX.
I guess 98% of the users have only the GPS connected and no other serial use
device so you can stick to the first one available (0).
The others have to give the path by hand.

Perhaps there is another way to get the path/device ID out of
/proc/bus/usb/devices, but I don't know that.
