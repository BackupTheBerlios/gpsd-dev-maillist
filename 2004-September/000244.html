That's not very standard conform!
On a normal Linux system you would use
/etc/sysconfig/gpsd
Content:
# Description ....
TTY="/dev/ttyS0"
# Description ....
BAUD_RATE="9600"

And in the init script do:
. /etc/sysconfig/gpsd
gpsd -p $TTY -s $BAUD_RATE -K
