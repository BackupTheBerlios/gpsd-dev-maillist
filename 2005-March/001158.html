jebus, is my math ever bad this morning. 0x32 means:
b0 = 0 = data fields invalid
b1 = 1 = clear ephemeris
b2 = 0 = don't clear history
b3 = 0 = no factory reset
b4 = 1 = enable nav lib data
b5 = 1 = enable debug data

strictly speaking, you probably don't need to clear the ephemeris, nor
do you need to enable debug data. so a reset config of 0x10 is probably
sufficient.
