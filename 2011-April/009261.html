Critical Issue is that in the Program("gpsd" ...) line it specified a
LINKFLAGS which replaces all exisiting flags

if we want to preserve existing flags then the following is needed instead

gpsd_env = env.Clone()
gpsd_env.MergeFlags("-pthreads")
gpsd = gpsd_env.Program("gpsd", LIBS="...")
