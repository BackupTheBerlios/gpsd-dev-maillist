here is a patch that adds PRN translation between SBAS and NMEA satellites
numbers. I assume that binary protocols report satellites as SBAS PRNs. GPSD
native message will report satellites as SBAS. I don´t have access to NMEA
documentation but documentation of GPS with WAAS support I met describe
translation like this. I assume this is part of NMEA specification.
