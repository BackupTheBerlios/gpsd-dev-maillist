   Not a definitive answer by any means, but:

./autogen.sh
sed -i '4800,4900 s/^ac_python=yes$/ac_python=no/' configure
./configure

   yields:

        Client Features
        ---------------
Old protocol in libgps  : yes
Build ncurses programs  : yes
Enable Python support   : no
Enable C++ support      : yes
Enable Qt support       : yes
- ------------------------------------------

then "make && make install" appear to perform as expected. "make check"
will not work, but that's also expected.
