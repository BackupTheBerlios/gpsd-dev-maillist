Andreas Brauchli <a.brauchli at elementarea.net>:
> just out of curiosity, how do i get the sirf version my device is using?
> is there a possibility to upgrade it (using the USB port)? i haven't
> seen firmwares laying around so i guess not, but maybe i'm wrong.

Run at -D 4.  You'll see a firmware versioin message in the log.

It is possible to upgrade the firmware through the USB port using
images from the SiRF site, but there are at present no upgrade tools
that run under Linux or *BSD.

The SiRF FAE I'm in contact with has given me Windows source code 
for an updater.  I thought my friend Rob Landley was going to use
this to write a Linux one, but he just got a new job so this is
looking unlikely.

If you want to take this on, I'll give you a copy.
