 I was making another patch for emul when I came across this:

/*
 * The name of a tty device from which to pick up whatever the local
 * owning group for tty devices is.  Used when we drop privileges.
 */

#define PROTO_TTY "/dev/ttyS0"

That works fine on linux, but you won't find that node on darwin or bsd.
I propose chaning this definition to "/dev/tty", as this seems to have the
same group set as the serial nodes in what I have seen so far.  Anyone?
