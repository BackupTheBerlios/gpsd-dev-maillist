I repatched against 2.23, the patch now includes also updated docbooks,
so detailed documentation is provided.

A few notes:
I had to change return value of gps_poll, because if read() returns 0 it
does not mean it had nothing to read (read is a blocking function!), but
that the socket have been closed, so the gpsd connection won't be
available anymore.

I also added a new shared library: libgpsmm, this library is the C++
wrapper for the libgps.

All other info on functions I added are documented in docbooks.
