If I understood your problem, I suggest you to use gpsfake (already in
the gpsd package). It takes some dummy data from a "log" file and feed
the gpsd daemon.
To generate that log file you may use gpsfeed+ (not in the gpsd
package), but it uses X (you may still generate the file on another box,
and then copy it to your MIPSEL).
I'm also attaching such a log file, so you can start testing your
program right now!
