Heya,

this mail goes to several upstreams/devel lists of packages I maintain
in Debian, so please excuse if it sounds impersonal.

On Thursday, December 13th Ubuntu will have their 'Debian Import Freeze'
[1] for Hardy, their upcoming release.

If you'd like to see any changes in the current Debian package before it
is frozen in Ubuntu it needs to be uploaded to Debian _before_ December
3rd to make sure it'll have 10 days to migrate to Debian/testing. As
migrating often takes longer than 10 days, it should be uploaded as soon
as possible.
So if you have something that should go into Ubuntu Hardy, please
contact me as soon as possible.


Thanks & cheers,

Bernd
